# Instant Wellness Kits

A React 19 + TypeScript frontend for Instant Wellness Kits, built with Vite, TanStack Router, MUI, Stripe, and form/validation tooling.

Also read API docs: **[Backend README.md](https://github.com/Clan-Codeprano-INT20H-26/backend/blob/main/README.md)**

## Tech Stack

| Category          | Choice                                            |
| ----------------- | ------------------------------------------------- |
| **Runtime**       | React 19, TypeScript 5.9                          |
| **Build**         | Vite 7, SWC (React)                               |
| **Routing**       | TanStack Router (file-based, auto code-splitting) |
| **UI**            | MUI (Material UI) 7, Emotion                      |
| **Forms**         | React Hook Form, Zod, `@hookform/resolvers`       |
| **HTTP**          | Axios                                             |
| **State**         | Zustand                                           |
| **Payments**      | Stripe (Elements, Checkout)                       |
| **Maps**          | Mapbox GL, react-map-gl                           |
| **Feedback**      | react-hot-toast                                   |
| **Lint / Format** | ESLint 9 (flat config), Prettier                  |
| **Git**           | Husky + lint-staged (pre-commit)                  |

- **Routing**: File-based with TanStack Router. Add routes under `src/routes/`; layout routes use `(authLayout)` and `(profileLayout)`; `routeTree.gen.ts` is regenerated by the Vite plugin.
- **Path alias**: `@/*` → `src/*` (see `tsconfig.app.json` and `vite-tsconfig-paths`).

## Responsiveness

The UI is responsive on **laptop** and **desktop** viewports. Layout and spacing adapt for comfortable use on typical laptop and desktop screens.

## Theming

- **Palette**: Custom MUI palette in `src/styles/palette.ts` (primary indigo, secondary light slate, semantic error/warning/info/success, grey/blue/green scales).
- **Typography**: Work Sans–based theme in `src/styles/typography.ts`.
- Theme is applied via `ThemeProvider` in `src/providers/ThemeProvider.tsx`; root layout also wraps the app with `StripeProvider` and `Toaster`.
